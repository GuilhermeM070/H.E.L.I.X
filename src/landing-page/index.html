Sobre o Projeto

H.E.L.I.X. — Handler Eficiente para Log de Integração e eXtração.
É um Projeto Python de extração automatizada de vídeos com foco em modularidade, escalabilidade e usabilidade futura via GUI.
A base utiliza yt_dlp e ffmpeg, permitindo baixar mídias com qualidade personalizada, suporte a áudio/vídeo separados e lógica preparada para expansão.

Funcionalidades

  • Download inteligente
  • Conversão automatizada
  • Organização de pastas
